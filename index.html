Este Portfólio tem o objetivo de colocar em prática todos os conhecimentos adquiridos ao longo dos três módulos de HTML5 e CSS3 do Curso em Vídeo, bem como o curso de JavaScript e PHP. Ao longo do site explicarei detalhadamente os objetivos de cada comando, tag e função assimilada durante os cursos feitos.
É de ressaltar que durante o meu período acadêmico na Universidade A Politécnica, aprendi a linguagem de programação Java e a utilizar a linguagem de manipulação de banco de dados MySQL.

Formatos de Imagens

JPEG - Formato de imagem que ocupa menor memória de compactação.

PNG - Formato de imagem que ocupa maior memória de compactação e permite transparência.

Favicon

Para utilizar um favicon é necessário usar a tag `link rel="shortcut icon" href="imagem/favicon.ico" type="image/x-icon"`.

Hierarquia de Títulos.

Semantica

Cada tag tem o seu significado; mesmo sendo colocada no princípio ou no fim no HTML, o significado da tag é que manda.

Formatação de texto

Aprendi a formatar o texto de forma a colocar um texto em negrito, em itálico, marcar o texto, texto deletado, texto sublinhado e uma expressão simples: x20+3=0, colocando abbr
Também como formatar código usando a tag 'pre' e 'code'. Como por exemplo:

                
                    int n1 = 2;
                    int n2 = 3;
                    int s = 0;

                    System.out.println("A soma de "+n1+"+"+n2+"e igual a: "+s);
