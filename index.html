Projeto Portfólio de Estudos

por: Willton Da Costa Xavier

Este Portfólio tem o objetivo de colocar em prática todos os conhecimentos adquiridos ao longo dos três módulos de HTML5 e CSS3 do Curso em Vídeo, bem como o curso de JavaScript e PHP. Ao longo do site explicarei detalhadamente os objetivos de cada comando, tag e função assimilada durante os cursos feitos.
É de ressaltar que durante o meu período acadêmico na Universidade A Politécnica, aprendi a linguagem de programação Java e a utilizar a linguagem de manipulação de banco de dados MySQL.

Semantica

Imagens Dinamicas

Utilizando a tag 'pictures' e a tag 'source:m' pode se criar imgens dinamicas

Audio

Utilizando a tag 'audio' pode se colocar audiono site, outra maneira e utilizando a tag 'source:src' dentro da tag 'audio'

Video

Utilizando a tag 'video' pode se colocar um video no site, para ver se o navegador e compativel com o video utiliza se a tag 'source:src' com os varios tipos de videos e o navegador ira escolher um para carregar.

Modulo 2

Cores

Pode se colocar cores de varias maneiras como por exemplo, usando o 'background-color:cor', 'background-color:#000000', 'background-color:rgb(00,00,255)' , 'background-color:hsl(matiz,saturacao,luminosidade)' e 'background-image: linear-gradient(directon, cor1, cor2 e cor3)'

Fontes

Utilizando a shorthand 'font:font-style, font-weight, font-size e font-family'

Pode se importar fontes com '@import url()' e pode se anexar fontes baixadas com '@font-face{}'

Seletores Personalizados

Pode se selecionar ou identificar uma tag com id em css com o comando '#nomedoid' e classes com '.nomeclasse'

Utilizando psedoclasses com ':' e psedoelementos com '::'

Modulo 3

Background Image

Pode se colocar uma imagem de fundo com a shorthand 'background:cor, imagem, posicao, repeat, [size], attachtment'

Pode se tambem fazer um efeito Paralaxe

Tabelas

Utilizando a tag 'table' pode se criar uma tabela

Cada tabela tem coluns e linhas e pode se criar utilizando a tag 'tr' e 'td'

Pode se identificar a cabeca, o corpo e o pe de uma tabela usando as tags 'thead', 'tbody' e 'tfoot'

Pode se colocar legendas numa tabela com a tag 'caption'

Pode se identificar o escopo da tabela com o comando 'scope'

Utilizando o comando 'overflow' pode fazer com que as tabelas sejam responsivas

Por fim pode agrupar as tabelas com a tag 'colgroup'
